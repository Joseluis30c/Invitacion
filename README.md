# 📖 Descripción

**Invitación** es una aplicación web sencilla diseñada para mostrar una invitación digital para eventos (como bodas, cumpleaños, aniversarios o celebraciones especiales).

La página está diseñada para ser:

- Visualmente atractiva
- Fácil de compartir
- Compatible con dispositivos móviles
- Personalizable para diferentes eventos
  
Este proyecto puede servir como plantilla base para invitaciones digitales.

# ▶️ Cómo ejecutar el proyecto

### 1️⃣ Clonar el repositorio

``` bash
git clone https://github.com/Joseluis30c/Invitacion.git
```

### 2️⃣ Entrar a la carpeta del proyecto

``` bash
cd Invitacion
```

### 3️⃣ Abrir el archivo HTML

Solo abre el archivo en tu navegador:

    index.html
